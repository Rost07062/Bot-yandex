Бот для подготовки к ЕГЭ по русскому
============

Описание
------------
Этот бот помогает учащимся отрабатывать часто встречающиеся задание из банка ФИПИ ЕГЭ, чтобы отработать тему и подготовиться к экзамену.

Установка
------------
Чтобы установить бота, нужно выполнить несколько шагов:

1. Клонируйте репозиторий: `git clone https://github.com/Rost07062/Bot-yandex`
2. Зайдите в корень проекта: `cd Bot-yandex`
3. Установите библиотеки, остальные зависимости подтянутся автоматически: `pip install -r requirements.txt`
4. В файле config.py поставьте свой токен в переменную BOT_TOKEN

Использование
------------
После запуска бота вы можете перейти в соответстующий диалог и прописать команду /start
